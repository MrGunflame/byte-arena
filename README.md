# byte-arena

no_std, no_alloc arena for allocating byte buffers.

## Usage

Add `byte-arena` to your `Cargo.toml`:
```toml
byte-arena = "0.1.0"
```

Refer to [docs.rs](http://docs.rs/byte-arena) for the usage of this crate.
